# TrackCovid
Android Application with Java integrated to Sri Lankan COVID API to fetch and display data

## API Used
Uses the public COVID 19 API offered by Sri Lanka

## Styles
Used fragments to display the data generated by API

## Libraries Used
Material Ui, Volley

## API Endpoint
https://hpb.health.gov.lk/api/get-current-statistical
